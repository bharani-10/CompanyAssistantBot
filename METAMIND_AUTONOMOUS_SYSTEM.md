## 🚀 HOW TO USE

### **Step 1: Launch MetaMind**
```bash
# Backend
cd backend && python -m uvicorn app.main:app --reload

# Frontend  
cd frontend && npm run dev
```

### **Step 2: Access Interface**
- Open: **http://localhost:3000**
- Navigate to **Mission Control**

### **Step 3: Autonomous Generation**
1. **Upload CSV dataset** (or use demo data)
2. **Describe your vision** (or use example prompts)
3. **Click "Launch Autonomous Generation"**
4. **Watch Agent Theater** for real-time progress
5. **Download complete project** from Project Center

### **Step 4: Deploy Anywhere**
- **Local**: `docker-compose up`
- **Hugging Face**: Upload generated `app.py`
- **Railway/Render**: Connect GitHub repository
- **Custom**: Use generated Dockerfile
